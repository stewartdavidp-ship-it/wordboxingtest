Word Boxing PWA v1.0.2 Release Notes
=====================================
Released: January 22, 2026

ABOUT WORD BOXING:
------------------
Multiplayer word battle game! Challenge friends to real-time
word fights. Build words, score points, knock out your opponent!

PWA FEATURES (NEW):
-------------------
- Installable as standalone app on iOS/Android/Desktop
- Works offline (single player mode)
- Auto-updates when new version available
- Install banner with iOS Safari instructions
- Detects iOS vs Android and shows appropriate install steps
- Banner dismisses for 24 hours when closed

GAME FEATURES:
--------------
- Real-time multiplayer via Firebase
- Create or join match rooms
- Build words from shared letter pool
- Scoring based on word length and rarity
- Round-based gameplay with knockouts
- Solo practice mode available
- Dark/light mode support
- Share results with match stats
- Game Shelf integration via GS badge

DEPLOYMENT:
-----------
Upload to web server root (HTTPS required):
  /index.html
  /manifest.json  
  /sw.js
  /icons/icon-*.png (all sizes)

FIREBASE SETUP:
---------------
Word Boxing requires Firebase Realtime Database for multiplayer.
Configure your Firebase project in the HTML file.
